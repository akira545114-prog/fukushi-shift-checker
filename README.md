# 指定障害児通所支援事業人員配置チェックシステム

## 概要
指定障害児通所支援事業者が、省令等で定められている人員配置基準を満たしているかを
簡易的に確認できるWebアプリです。

## 作成背景
指定障害福祉サービスの指定及び指導監査等に携わる指定都市職員として、
省令等で定められている事業者が遵守すべき基準が複雑であり、各事業者の
制度理解の不足やそれに伴う事業者側及び行政側の業務負担が課題と感じました。
その中でも、指定障害児通所支援事業の人員配置は、事業運営の中で最も重要な
ポイントであり、事業者の業務負担の軽減や基準違反による給付費返還の
リスク回避を目的に本Webアプリを作成しました。

## 主な機能
- 指定障害児通所支援事業の人員配置基準チェック
- 一般的な定員10名の事業所想定
- 各日における基準人員の充足／不足の判定表示
- 児童発達支援管理責任者の常勤要件の可否の判定表示

## 使用技術
- HTML
- CSS
- JavaScript（ES6）

## 工夫した点
- 法令上の複雑な条文を読むことなく、事業者が視覚的に人員配置基準を判断できる点
- JavaScriptで条件分岐を用い、判定ロジックを実装した点
